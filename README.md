## basics

ğŸ‘‰ bx.bitnats are [ordinal inscriptions](https://docs.ordinals.com/) on Bitcoin  
ğŸ‘‰ bx.bitnats are on-chain claims to Bitcoin block data using the [tap protocol](https://github.com/Trac-Systems/tap-protocol-specs)  
ğŸ‘‰ bx.bitnats are derived from Digital Matter Theory [DMT](https://digital-matter-theory.gitbook.io/digital-matter-theory)  
ğŸ‘‰ bx.bitnats are generated each time a Bitcoin block is mined  
ğŸ‘‰ bx.bitnats are available to mint on [mscribe.io](https://mscribe.io/nats?t=latest)
